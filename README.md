<h1 align="center">PokeClicker</h1

PokeClicker is a solo project created by Randy Hac, inspired by Cookie Clicker. Select one of the three iconic starter Pokemon (Charmander, Squirtle, and Bulbasaur) and begin clicking to generate experience, level up their stats, and watch them Evolve! 

## Technologies Used
PokeClicker was built using the following technologies:

### Backend:
- **Javascript**
- **MongoDB**
  - mongoose-express
